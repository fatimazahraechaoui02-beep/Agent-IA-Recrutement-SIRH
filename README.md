# Agent IA de Tri de CV - Chargé(e) de Recrutement

## 📌 Présentation de la Mission
Dans le cadre d'un processus de recrutement digitalisé, ce projet consiste à concevoir un prototype d'Agent Intelligent capable d'automatiser la phase de présélection. L'agent analyse, traite et classe **15 CVs** pour le poste de "Chargé(e) de Recrutement - Secteur Automobile".

## 🛠️ Choix Technologiques
- **Plateforme Coze :** Utilisée pour le développement de l'Agent IA (No-Code).
- **Modèle LLM :** Pour l'analyse sémantique et l'extraction des features.
- **GitHub :** Pour le dépôt organisé et documenté du projet.

## 📂 Structure du Projet
- `/data` : Contient le jeu de données (15 CVs au format PDF).
- `/instructions` : Contient la logique de scoring et le prompt de l'Agent.
